# Bridge Competition Dashboard

## Overview
The **Bridge Competition Dashboard** is a web-based system designed to manage, score, and visualize results for the **Union College of Union County, NJ (UCNJ)** annual Bridge Building Competition.  

It supports multiple bridge types: **Truss** and **Drawbridge**, and provides live ranking and analytics for judges and faculty during the event.

The platform was **deployed and used in the 2025 competition**, supporting real-time scoring and participant engagement.

---

## Features
- **Automated Scoring** - Calculates and ranks bridge results dynamically from uploaded or input JSON data.  
- **Real-Time Leaderboards** - Displays ranked results by bridge type and overall performance.  
- **Interactive UI** - Simple, responsive design built for live use during competition events.  
- **JSON-Based Data Flow** - Lightweight and portable format for results management.  
- **Event Tested** - Successfully used at UCNJ’s Bridge Competition for student engineering events.

---

## Project Purpose
This project was developed to modernize and simplify how UCNJ’s annual engineering competition tracks and displays performance.  
It served as an **applied computing project** connecting engineering design, data science, and software development.

---

## Technologies Used
- **JavaScript, HTML, CSS** - Front-end visualization and UI  
- **JSON** - Data handling and score input  
- **GitHub Pages / Local Hosting** - For event-day deployment
